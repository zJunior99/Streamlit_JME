# 📊 Smart Sentiment Dashboard

An interactive NLP web application built with **Python**, **Streamlit**, and **TextBlob**. This tool analyzes the sentiment of text inputs in real-time, providing polarity and subjectivity metrics while maintaining a historical comparison of all entries.

## 🔗 Project Links
*   **Live Application:** [View Demo](https://my-app-stream-jme.streamlit.app/)
*   **Detailed Article:** [Read on Medium](https://medium.com/@jm2022075474/from-forgotten-scripts-to-living-dashboards-the-streamlit-revolution-in-data-analysis-1643b5ff98e6)

## ✨ Key Features
*   **Real-time Sentiment Analysis:** Instant scoring of polarity and subjectivity.
*   **Dynamic Visualizations:** Bar charts comparing historical entries using `st.session_state`.
*   **Clean UI:** Responsive dashboard design with color-coded status alerts.
*   **Automated Deployment:** Integrated with Streamlit Community Cloud for CI/CD.

## 🛠️ Tech Stack
*   **Language:** Python 3.x
*   **Framework:** Streamlit
*   **NLP Library:** TextBlob
*   **Data Visualization:** Native Streamlit Charts

## 📦 Installation & Local Execution

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git)
    cd YOUR_REPO_NAME
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the app:**
    ```bash
    streamlit run app.py
    ```

## 📂 Project Structure
*   `app.py`: Main application logic and UI.
*   `requirements.txt`: List of necessary Python libraries (streamlit, textblob).
*   `README.md`: Project documentation.
